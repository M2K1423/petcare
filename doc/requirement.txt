1. Mục tiêu hệ thống
•	Quản lý toàn bộ hoạt động phòng khám thú y: 
o	hồ sơ thú cưng 
o	lịch khám 
o	bệnh án 
o	tiêm phòng 
o	thanh toán 
•	Hỗ trợ: 
o	bác sĩ thú y 
o	lễ tân 
o	admin 
•	Có thể mở rộng: 
o	AI tư vấn thú y 
o	nhắc lịch tự động 

2. Actors (Tác nhân)
Pet Owner (chủ thú cưng – portal web)
Vet (bác sĩ thú y)
Receptionist (lễ tân)
Admin

3. FLOW HỆ THỐNG
Pet Owner Flow (web portal)
1.	Đăng ký / đăng nhập 
2.	Tạo hồ sơ thú cưng: 
o	tên 
o	loài 
o	tuổi 
3.	Đặt lịch khám 
4.	Nhận thông báo lịch hẹn 
5.	Xem: 
o	bệnh án 
o	lịch tiêm phòng 
6.	Thanh toán 

Vet Flow
1.	Đăng nhập 
2.	Xem lịch khám 
3.	Xem hồ sơ thú cưng 
4.	Nhập: 
o	triệu chứng 
o	chẩn đoán 
o	thuốc 
5.	Cập nhật bệnh án 

Receptionist Flow
1.	Tạo / quản lý lịch hẹn 
2.	Check-in thú cưng 
3.	Tạo hồ sơ mới 
4.	Thanh toán 

 Admin Flow
1.	Quản lý user 
2.	Quản lý bác sĩ 
3.	Quản lý dịch vụ 
4.	Xem báo cáo
